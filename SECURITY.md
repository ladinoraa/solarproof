# Security Policy

Report vulnerabilities via [GitHub Security Advisories](../../security/advisories/new) — do not open public issues.

We acknowledge within 48 hours and aim to fix critical issues within 14 days.

## Scope
- Smart contracts (`apps/contracts/`)
- API routes and meter signature verification
- Minter key management

## Out of scope
- Stellar network issues → [Stellar bug bounty](https://stellar.org/bug-bounty)
